# DSA Java Practice Repository

Welcome to the **DSA Java Practice Repository**! This repository is dedicated to practicing Data Structures and
Algorithms (DSA) in Java. Below, you will find an organized index of problems and questions implemented in this
repository.

## Table of Contents

1. [Average of Array Elements](src/tech/kaustubhdeshpande/array/AverageOfArrayElements.java)

## How to Use

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Open the project in your favorite IDE (e.g., IntelliJ IDEA).
3. Navigate to the `src` folder to explore the solutions.
